## Definitions

- **AMCore version**: the highest `AM-IMP` version AMCore fully packages in `versions/`.
  This is "what version AMCore itself is".
- **Latest tracked improvement**: the highest `AM-IMP` version known to any source
  (registry or a project manifest). This is "the current version of the family".
- AMCore is **current** when it packages every tracked version up to the latest.
