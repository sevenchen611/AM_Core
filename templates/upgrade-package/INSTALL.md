# Install

## Preconditions

- Current project is identified.
- Current project `.env` exists.
- Current project has its own Notion, LINE, Render, and GitHub settings.

## Steps

1. Read `upgrade.json`.
2. Check required env vars.
3. Create or verify required Notion databases.
4. Install required scripts.
5. Apply code changes.
6. Update `.env.example` and deployment checklist.
7. Run verification.
8. Update project manifest.
9. Create project upgrade record.
