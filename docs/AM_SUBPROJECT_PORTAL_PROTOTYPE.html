AM User UI

登入後依權限查看專案環境設定、可安裝版本與開發者設定。

Prototype 帳號：admin / admin-demo、seven / seven-demo、hozo / hozo-demo。正式版本應改用後端驗證、Session 與權限表。

AMCore 管理每個子專案的安裝脈絡

子專案使用者只看自己的環境資料與可安裝版本。開發者可以查看全部專案、調整設定、確認版本是否開放安裝。

LINE 對話群組與附件

LINE 群組是議題串的入口；訊息、附件、檔案解析與任務判斷都應能從這裡回查。

會議紀錄

會議紀錄同時是任務來源與執行知識；checkbox 項目會直接成為確認任務。

任務判斷規則

AM Core 判斷規則是共用基礎；專案判斷規則是各子專案依任務校準累積的補充。

手動加入任務判斷規則

原型會先加入目前專案規則；正式版寫回專案本地判斷規則庫。

適合放進使用者入口的 Database

可安裝版本
