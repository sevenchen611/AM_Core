# Install

1. Create or verify the project's own responsibility databases.
2. Install `setup-responsibility-owner-narrowing.js` if the project does not already have a setup script.
3. Install `sync-responsibility-candidates.js`.
4. Add project-local env variable names to `.env.example` and Render checklist.
5. Run a dry run before writing.
6. Run write sync only after dry run is valid.
7. Update manifest and upgrade record.

Reference scripts are stored in `scripts/`.
