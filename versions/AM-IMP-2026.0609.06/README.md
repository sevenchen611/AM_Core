# AM-IMP-2026.0609.06 - Task judgment rules User UI

This upgrade makes task judgment rules visible in the User UI.

The rules page is labeled `任務判斷規則` and shows:

- AMCore shared task judgment rules.
- Project-specific SevenAM or HOZO AM rules.
- Learned calibration rules from the current project only.

The goal is to make the morning/hourly task judgment logic visible and auditable without mixing project data.
