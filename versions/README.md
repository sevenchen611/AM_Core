# Versions

Store one complete upgrade package per folder.

Example:

```text
versions/
  AM-IMP-2026.0608.08/
    README.md
    upgrade.json
    INSTALL.md
    REQUIRED_DATABASES.md
    ENVIRONMENT.md
    VERIFY.md
    ROLLBACK.md
    scripts/
    notion-schemas/
    patches/
```

Do not store project secrets or live customer data in version packages.
