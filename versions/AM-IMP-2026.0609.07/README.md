# AM-IMP-2026.0609.07 - Manual Task Judgment Rule Capture

This package adds a manual rule-capture area below the learned calibration rules on the User UI `任務判斷規則` page.

The form lets a project user write and save a new task judgment rule into that project's own judgment rules data source.

AMCore's own User UI prototype also displays this manual rule-capture area as the shared design source.

The saved rule is project-local. AMCore stores only the shared UI and control API pattern.
