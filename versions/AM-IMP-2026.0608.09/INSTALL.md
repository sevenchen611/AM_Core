# Install

1. Add immediate command parsing to `src/server.js`.
2. Use the project's own assistant triggers.
3. Remember recent task lists by LINE conversation context.
4. Add ordinal task detail lookup.
5. Add high-risk command acknowledgement.
6. Add health endpoint fields:
   - `immediateCommandEnabled`
   - `immediateCommandPrefixes`
7. Update manifest and upgrade record.
