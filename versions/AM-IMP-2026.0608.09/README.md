# AM-IMP-2026.0608.09

## Immediate LINE Command Conversation Mode

This package lets LINE messages that begin with the assistant trigger act as immediate commands.

Supported immediate behavior:

- Query current tasks.
- Remember the last task list in the conversation.
- Open task details by ordinal, such as "打開第 2 個任務".
- Acknowledge other low-risk commands and queue them for Codex handling.
- Avoid directly executing high-risk commands.
