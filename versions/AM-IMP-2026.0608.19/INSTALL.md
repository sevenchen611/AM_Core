# Install

1. Read the shared layout registry:

```text
D:\Codex_project\AM_Core\config\notion-view-layouts.json
```

2. Find the project-local total-control task database.

3. Update the project-local `Default view` table so it does not display:

```text
來源原文
```

4. If another project-local task review table displays `來源原文`, hide it there too.

5. Keep the property itself and all existing values.

6. Update the project-local `docs/project-improvement-manifest.md`.

7. Add a project-local upgrade note under `docs/upgrades/`.
