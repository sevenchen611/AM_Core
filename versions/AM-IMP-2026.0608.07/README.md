## Five-Slot Goal Recognition And Confirmation Workflow

This package standardizes the five scheduled report slots so each report reminds the operator to confirm completion goals before tracking progress.

## Required Slots

- 08:00 morning brief
- 10:00 follow-up
- 13:00 follow-up
- 17:00 follow-up
- 20:30 daily report
