# AM-IMP-2026.0708.01 — BuildAM 工程管理模組(第一期 F1–F5)

BuildAM(工程 AM)的完整第一期能力,開發於 2026-07-06 ~ 07-08,依據
`SPEC_工程管理系統_第一期_v1.2.md` 與 `SOP-01 設計圖審閱至發包交底 v1.0`。

## 能力範圍

- **F1 收集與初判**:LINE 群組訊息/照片 100% 落庫,群組路由(群組→專案+角色+工種),
  MiniMax-M3 AI 初判(空間/工項/類型/信心度,空間別名對照),兩層自動過濾
  (系統轉貼直接歸檔、高信心一般對話自動歸檔),確認佇列 Web UI
  (縮圖、雙向連帶掛載、佇列內新增工項、掛到回饋單、批次確認、審核修正記錄)。
- **F2 單據生命週期**:回饋單(自動編號、開單公告、回覆中、銷項強制佐證、
  擱置(待時機)+觸發工項自動復活+週一盤點)、變更單(核准/退回、未核准不得可施作)。
- **F3 照片歸檔**:原圖 Drive「未歸檔/日期」→ 確認後自動搬「專案/空間/工項/日期」;
  儀表板空間照片時間軸。
- **F4 儀表板**:專案卡片(未銷項/A級/逾期/擱置/待核准/動工倒數/推播額度)、
  Google 行事曆內嵌、CSS 甘特圖、待辦任務、SOP-01 階段檢核、會議清單、
  空間×工種矩陣、單據/會議彈窗檢視與就地編輯。
- **F5 雙向推播**:開單公告、每日到期/逾期提醒(真 @mention 點名)、逾期升級、
  手動催辦(預覽後發送)、變更公告、會議記錄發布;行程前一晚+30 分前提醒
  (GitHub Actions 每 15 分巡邏 /cron/reminders)。
- **會議錄音管線**:錄音傳群組 → Gemini 轉寫提煉 → 會議記錄(摘要/決議/checkbox 待辦/
  錄音回放連結)+ 待辦任務展開 → 摘要發布回群。
- **AM Portal 整合**:接受 rental.hozorental.com 簽發之 hozo_session 單一登入,
  依 am-buildam 與子專案授權(am-buildam-zs/hz/sys)於伺服器端過濾可視範圍。

## 與 AM 家族的關係

BuildAM 為最小種子專案(僅裝 0608.01 資料隔離、0608.05 manifest),
本包將其工程模組整包收錄,供未來新的工程型 AM 專案複製部署。
資料隔離 guard(assertBuildNotionTarget)較家族原版更嚴格:
以 env 宣告的 data source 白名單+母頁面驗證。
